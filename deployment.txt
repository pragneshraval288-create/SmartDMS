## âš™ï¸ Environment Configuration

### Create `.env` File

Create a file named `.env` in the **project root directory** with the following content:

```env
# Flask Configuration
SECRET_KEY=your_secret_key_here_generate_random_string

# Database Configuration
DB_USER=smartdms_user
DB_PASS=your_secure_database_password
DB_HOST=127.0.0.1
DB_PORT=3306
DB_NAME=smartdms_enterprise

# Encryption Configuration
SMARTDMS_ENC_KEY=your_fernet_encryption_key_here

# Optional: Development Mode
FLASK_ENV=development
FLASK_DEBUG=True         # Development only
```

### ğŸ” Security Guidelines

**Generate SECRET_KEY (Python):**
```python
import secrets
print(secrets.token_hex(32))
```

**Generate SMARTDMS_ENC_KEY (Python):**
```python
from cryptography.fernet import Fernet
print(Fernet.generate_key().decode())
```

> âš ï¸ **CRITICAL:** Never commit `.env` to version control. Add it to `.gitignore`.

## ğŸ—„ï¸ Database Setup

### Step 1: Start MySQL Server

**Windows:**
```cmd
net start MySQL80
```

**Linux:**
```bash
sudo systemctl start mysql
```

**macOS:**
```bash
brew services start mysql
```

### Step 2: Login to MySQL

```bash
mysql -u root -p
```

### Step 3: Create Database

```sql
-- Create database with proper character set
CREATE DATABASE smartdms_enterprise
    CHARACTER SET utf8mb4
    COLLATE utf8mb4_unicode_ci;
```

### Step 4: Create Dedicated User (Recommended)

```sql
-- Create user
CREATE USER 'smartdms_user'@'localhost'
    IDENTIFIED BY 'your_secure_database_password';

-- Grant privileges
GRANT ALL PRIVILEGES ON smartdms_enterprise.*
    TO 'smartdms_user'@'localhost';

-- Apply changes
FLUSH PRIVILEGES;

-- Exit MySQL
EXIT;
```

### Step 5: Verify Connection

```bash
mysql -u smartdms_user -p smartdms_enterprise
```

## ğŸ”§ Troubleshooting

### MySQL Connection Error
**Problem:** `Can't connect to MySQL server`

**Solutions:**
- Verify MySQL service is running
- Check database credentials in `.env`
- Ensure DB_HOST is `127.0.0.1` or `localhost`
- Verify firewall settings

### Module Not Found Error
**Problem:** `ModuleNotFoundError: No module named 'flask'`

**Solutions:**
- Ensure virtual environment is activated
- Reinstall dependencies: `pip install -r requirements.txt`
- Check Python version: `python --version`

### Environment Variables Not Loading
**Problem:** Application can't read `.env` file

**Solutions:**
- Verify `.env` file exists in project root
- Ensure `python-dotenv` is installed
- Check file encoding (should be UTF-8)
- Restart the application

### Port Already in Use
**Problem:** `Address already in use`

**Solutions:**
- Change port in `run.py`: `app.run(port=5001)`
- Kill process using port 5000
- Restart your computer

### Permission Denied (Windows)
**Problem:** Script execution is disabled

**Solutions:**
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
